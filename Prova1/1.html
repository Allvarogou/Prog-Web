Durante o acesso, quando o usuario coloca a url, é feita a solicitação de acesso via internet, onde se validado, o usuario consegue entrar no site. Podemos ver esta relação no modelo servidor cliente, onde o cliente faz a requisição de entrada para o site (servidor) e após a requisição ser autorizada ou não, o site é exibido. utilizando o protocolo http/https, podemos fazer o a requisição, onde o servidor recebera o ip do computador, e o validara.

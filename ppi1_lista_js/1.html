O que é JavaScript?

O JavaScript surgiu em 1995, em um contexto onde a Web era construída apenas com HTML e, no máximo, CSS, tornando as páginas estáticas, sem interatividade e com uma experiência de navegação limitada e pouco atraente.

O JavaScript é tipicamente executado no lado do cliente (browser) por meio de interpretadores específicos (como V8 ou SpiderMonkey). Contudo, com o surgimento do Node.js em 2009, o código JS também passou a ser usado no backend (servidor) para responder ao cliente

O código JavaScript é interpretado pelo navegador dentro de um ambiente de execução que lhe permite interagir com o HTML e CSS através da Árvore DOM. Esse ambiente é isolado por meio do objeto global window, limitando o acesso do script a recursos externos do sistema.

O ambiente de execução do JavaScript é representado pelo objeto global window. Dentro dele, temos objetos chave como document (que representa a Árvore DOM) , alert() (que exibe uma caixa de diálogo) e console (que representa o console JS, usado para debug via console.log()).
